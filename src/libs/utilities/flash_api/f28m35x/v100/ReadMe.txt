F28M35x is supported by the TI-wide F021 Flash API:

http://www.ti.com/tool/f021flashapi

To run the example included in this directory, download the F021 Flash API from
the above URL and extract the "Include", "Libraries", and "Source"
folders into the controlSUITE\libs\utilities\flash_api\f28m35x\ directory
such that the directory structure is:

\utilities\flash_api\f28m35x\
                       .....\example\
                       .....\Include\
                       .....\Libraries\
                       .....\Source\
                       .....\Readme.txt
